# Calculate me! – A simple web calculator
A clean, responsive calculator built with HTML, CSS, and JavaScript.
Supports basic arithmetic operations (+, -, *, /), percentage (%), clear (C), and evaluation (=).
The layout uses custom utility classes for centering and flexbox, making it easy to customize.
# Note: M+ and M- buttons are UI placeholders (no memory logic yet).
The calculator uses eval() for simplicity – ideal for learning or local use.
# Live demo – open index.html in any modern browser.
