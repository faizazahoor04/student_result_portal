📋 Meta WhatsApp Cloud API — One-Time Setup (3–5 days)
1
Create Meta Developer App — Go to developers.facebook.com → My Apps → Create App → choose Business → Add WhatsApp product. You instantly get a test number and token.
2
Copy your credentials — In App Dashboard → WhatsApp → API Setup, copy the Access Token and Phone Number ID. Paste them into META_ACCESS_TOKEN and META_PHONE_NUM_ID in app.py.
3
Create message template — Go to business.facebook.com → WhatsApp Manager → Message Templates → Create. Use: Category: Utility, Name: result_notification, Language: English. Template body:
Assalam o Alaikum!
Result of {{1}}:
Marks: {{2}}/{{3}}
Grade: {{4}}
Percentage: {{5}}%
See full result at: your-school-website.com
4
Wait for template approval — Meta reviews templates in 24–48 hours. For immediate testing, add your own phone as a "Test Recipient" in the Meta dashboard — no approval needed for test recipients.
5
Add parent phones to Excel — Open student_data.xlsx → Students sheet → add Column H titled "Parent Phone" → enter numbers like 03001234567 → save → restart python app.py.

How to Add Parent Phone Numbers
